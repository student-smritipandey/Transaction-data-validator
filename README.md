# Transaction Data Validator

A Flask-based web application for validating transaction datasets and generating quality reports.

## Features

* Phone Number Validation
* Date Validation
* Payment Mode Validation
* Duplicate Record Removal
* Error Report Generation
* Valid Records Export
* Invalid Records Export
* Large CSV File Splitting
* Interactive Dashboard
* Bootstrap Professional UI

## Tech Stack

* Python
* Flask
* Pandas
* Bootstrap 5

## Project Structure

TransactionValidator/

├── app.py

├── config/

├── templates/

├── validators/

├── utils/

├── reports/

└── requirements.txt

## How to Run

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

4. Open browser

http://127.0.0.1:5000

## Output Files

* clean_data.csv
* valid_records.csv
* invalid_records.csv
* error_report.csv
